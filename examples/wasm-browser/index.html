About COOP/COEP Headers

This application uses WebAssembly with multi-threading capabilities (enabled via wasm-bindgen-rayon). To support SharedArrayBuffer, the browser requires special CORS headers:

  • Cross-Origin-Opener-Policy: same-origin - Isolates the browsing context
  • Cross-Origin-Embedder-Policy: require-corp - Requires CORS for all resources

These headers are automatically set by the Vite development server and should be configured in your production environment (e.g., Nginx, Cloudflare, AWS).
